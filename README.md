# PrologVM

Project: C++ based Prolog Virtual Machine
 
Usage:
./go.sh add

Assuming Prolog folders are kept in progs/ folder.

Runs on Mac OS, Ubuntu & his flavors.

Mimicked the Java code available here: https://github.com/ptarau/iProlog
